# COUNCIL_WORKFLOW_FRAMEWORK

## Objective
Create the Council workflow framework in `AGENTS.md`, provide a configurable Council definition in `THE_COUNCIL.md`, and add any required supporting files.

## Acceptance Criteria
- The workflow described in the prompt is codified in `AGENTS.md`.
- The Council is configurable through `THE_COUNCIL.md`.
- There is a clear, easy mechanism to change Council size.
- Any required supporting files are added.
